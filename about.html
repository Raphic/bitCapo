What are Cryptocurrencies?

A cryptocurrency can be best described as a piece of digital software which makes use of a cryptographic algorithm which contains a database. This database is also known as the Blockchain.

Cryptocurrencies can be exchanged between persons without having the need for an intermediary (like a bank) to approve the transactions or having to hold your software.

The monetary value of a cryptocurrency comes from the exchange rate towards the traditional currencies such as the U.S. dollar and EURO.

By having an exchangeable person to person software with a cryptographic algorithm, there is also a need for security. This is where the blockchain database comes in.

The blockchain is often a public ledger which contains information about every transaction of data on its chain. All this data gets compiled into a separate file on the blockchain called blocks. How often the blockchain requires data verification determines how many blocks are created per second(s) or minute(s) and this is set within the parameters of the blockchain.

Starting from the first block, the following blocks gets added on every x amount of time, creating a chain of blocks.

Why is this secure?

Every block gets verified by multiple computers communicating with the database, if all the computers verified the block with the same results, then the block gets added to the database.

Through this way human errors are eliminated, the cost of this communication is electricity.

Why would these computers spend electricity costs to verify these blocks for the database?

If we look at cryptocurrencies most of them have a set amount of software that will ever be created, the verifying computers, also called miners, get rewarded for their work through a block reward.

What is a block reward?

A block reward is the amount of cryptocurrencies being released when a block is verified, the amount that is released gets rewarded (or paid) to the miners for their work. Through this way a database is sustainable.

What happens if the set amount, maximum supply, is reached?

Once the maximum supply of the software is reached the miners will get rewarded with the fee’s that are created for uploading data on the database. The fee on every data chain is different but in most cases it will be below $0.001 per upload. Adding all these fees up, should be enough incentive to keep working on updating and verifying the database.

If this is not the case then there are always other possibilities to discover, such as reducing the amount of software that is released every block.
